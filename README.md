# Level Up Fitness App

Level Up is a gamified fitness tracking app built with SwiftUI.

The goal of the app is to motivate users to stay consistent with workouts by incorporating game mechanics such as levels, leaderboards, and daily challenges.

## Features

- Start / login screen
- Workout tracking dashboard
- Weekly progress charts
- Daily workout challenges
- Leaderboards to compete with friends
- User profile and achievements

## Built With

- Swift
- SwiftUI
- Xcode
- Git & GitHub
